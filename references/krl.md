# KRL Commuter Line (KAI Commuter) + KA Bandara

Heavy-rail commuter network across Jabodetabek + Merak. Source: KAI Commuter
Jabodetabek & Merak route map (see `../assets/krl-map.png`). **Manggarai** is the
central interchange for most lines.

> Station ordering below is terminus-to-terminus. Lines share track through the
> central area, so transfers happen at Manggarai, Tanah Abang, Duri, Jatinegara,
> Kampung Bandan, Jakarta Kota.

---

## Lin Bogor (Red) — Jakarta Kota ↔ Bogor / Nambo
**Jakarta Kota** → Jayakarta → Mangga Besar → Sawah Besar → Juanda → Gondangdia →
Cikini → **Manggarai** → Tebet → Cawang → Duren Kalibata → Pasar Minggu Baru →
Pasar Minggu → Tanjung Barat → Lenteng Agung → Universitas Pancasila →
Universitas Indonesia → Pondok Cina → Depok Baru → Depok → **Citayam** → Bojong
Gede → Cilebut → **Bogor**

**Nambo branch:** splits at **Citayam** → Pondok Rajeg → Cibinong → **Nambo**

(Trains also run Bogor ↔ Manggarai/Angke/Kampung Bandan via the loop.)

---

## Lin Cikarang (Blue) — Cikarang ↔ central (loop line)
Operates as a loop: from Cikarang to Jatinegara, then two ways around central
Jakarta (via Manggarai *or* via Pasar Senen), forming the **Cikarang Loop Line**.

**Cikarang → Jatinegara:**
Cikarang → Metland Telaga Murni → Cibitung → Tambun → Bekasi Timur → Bekasi →
Kranji → Cakung → Klender Baru → Buaran → Klender → **Jatinegara** (junction)

**From Jatinegara, two routes around the loop:**
- *Via Manggarai (south arc):* Jatinegara → Matraman → **Manggarai** → Sudirman →
  Karet → **Tanah Abang** → Duri → Angke → **Kampung Bandan**
- *Via Pasar Senen (north arc):* Jatinegara → Pondok Jati → Kramat → Gang
  Sentiong → Pasar Senen → Kemayoran → Rajawali → **Kampung Bandan**

**Tanah Abang** → Palmerah → Kebayoran → Pondok Ranji → Jurang Mangu → Sudimara →

## Lin Tangerang (Brown) — Duri ↔ Tangerang
**Duri** → Grogol → Pesing → Taman Kota → Bojong Indah → Rawa Buaya → Kalideres →
Poris → Batu Ceper → Tanah Tinggi → **Tangerang**

## Lin Tanjung Priok (Pink) — Jakarta Kota ↔ Tanjung Priok
**Jakarta Kota** → Kampung Bandan → Ancol → JIS → **Tanjung Priok**
*(JIS = Jakarta International Stadium; ↔ Transjakarta Kor.14.)*

---

## KA Bandara (Airport Train) — Manggarai ↔ Soekarno-Hatta
**Manggarai** → BNI City (Sudirman Baru) → Duri → Batu Ceper → **Bandara
Soekarno-Hatta**. Separate ticket from regular KRL. At the airport, connect to
the **Kalayang** people-mover for the terminals (below).

### Kalayang Bandara (Skytrain Soekarno-Hatta)
Free automated people-mover (APM) inside the airport, linking:
**Terminal 1 ↔ Terminal 2 ↔ Terminal 3 ↔ Stasiun KA Bandara (Integrated
Building)**. Use it to reach your terminal after arriving by KA Bandara. Not a
commuter line — airport-internal shuttle.

---

## KA Lokal Merak (codes LM1–LM11) — Rangkasbitung ↔ Merak
**Rangkasbitung** (LM1) → Jambubaru (LM2) → Catang (LM3) → Cikeusal (LM4) →
Walantaka (LM5) → Serang (LM6) → Karangantu (LM7) → Tonjongbaru (LM8) → Cilegon
(LM9) → Krenceng (LM10) → **Merak** (LM11)
*(Local train, not KRL Commuter. Order verified from the official Jabodetabek &
Merak map. Connects to Lin Rangkasbitung at Rangkasbitung = R22/LM1.)*

---

## Major KRL interchanges
- **Manggarai** — Bogor ↔ Cikarang ↔ KA Bandara; main transfer hub.
- **Tanah Abang** — Rangkasbitung origin ↔ Cikarang loop ↔ Transjakarta.
- **Duri** — Tangerang line ↔ Cikarang loop ↔ KA Bandara.
- **Jatinegara** — Cikarang loop branch split.
- **Kampung Bandan** — Cikarang loop ↔ Tanjung Priok ↔ Bogor (Jakarta Kota arc).
- **Jakarta Kota** — Bogor ↔ Tanjung Priok ↔ Transjakarta Kor.1.
- **Citayam** — Bogor main line ↔ Nambo branch.
- **Sudirman / BNI City** — ↔ MRT Dukuh Atas, LRT Jabodebek, Transjakarta.
