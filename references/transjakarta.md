# Transjakarta — BRT Corridors & Haltes

World's longest BRT network. Service families: **BRT** (corridor buses, dedicated
lanes), **Non-BRT / Angkutan Pengumpan** (feeder, mixed traffic — see
`transjakarta-feeder.md`), **Mikrotrans** (Jak Lingko microbus — same file),
**Royaltrans**, **Transjabodetabek**, **Bus Wisata**, **Transjakarta Cares**.

> Halte lists below are full corridor stop sequences (≈ June 2026). Endpoints and
> some stops shift during MRT/LRT construction (noted inline). Verify live in the
> **Tije** app.

---

## Koridor 1 — Blok M ↔ Kali Besar (22 halte)
Blok M → ASEAN & Kejaksaan Agung → Masjid Agung → Bundaran Senayan → Senayan Bank
DKI → Polda Metro Jaya → Bendungan Hilir → Karet → Dukuh Atas → Tosari → Bundaran
HI Astra → M.H. Thamrin → Kebon Sirih → Monumen Nasional → Harmoni → Sawah Besar
→ Mangga Besar → Taman Sari → Glodok → Kota → Museum Sejarah Jakarta → Kali Besar
*(parallels MRT North–South along Sudirman–Thamrin)*

## Koridor 2 — Pulo Gadung ↔ Monumen Nasional (23 halte)
Pulo Gadung → Bermis → Pulo Mas → Perintis Kemerdekaan → Pedongkelan → Cempaka
Mas → Sumur Batu → Cempaka Baru → Pasar Cempaka Putih → Rawa Selatan → Galur →
Senen Toyota Rangga → Senen Raya → RSPAD → Pejambon → Gambir → Istiqlal → Juanda
→ Pecenongan → Kwitang → Gambir 2 → Balai Kota → Monumen Nasional
*(rerouted from Harmoni to Monas since Mar 2023, MRT Phase 2A works)*

## Koridor 3 — Kalideres ↔ Monumen Nasional (14 halte)

Jembatan Gantung → Taman Kota → Damai → Jelambar → Grogol → Roxy → Petojo →
Monumen Nasional *(originally to Pasar Baru; that leg now served by Kor.8)*

## Koridor 4 — Pulo Gadung ↔ Galunggung (≈18 halte)
